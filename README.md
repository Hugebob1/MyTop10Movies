## Features

- **Rate and review** your favorite movies
-  **Delete** movies from your list
- **Sorts automatically** by rating (from highest to lowest)

## Tech Stack

- Python 3
- Flask + Flask-Bootstrap
- SQLAlchemy + SQLite
- Flask-WTF + WTForms
- TMDb API (https://www.themoviedb.org/)
